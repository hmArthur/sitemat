ESTUDO SOBRE HTML

HTML (HyperText Markup Language) é uma linguagem de marcação utilizada para criar uma estrutura e o conteúdo básico de páginas da web. Ela utiliza uma série de elementos (ou "tags") para marcar e formatar diferentes partes do conteúdo de uma página. Algumas das principais tags HTML incluem: 1. `html`: Define o início do documento HTML. 2. head`: Contém informações sobre o documento, como o título da página e metadados. 3. `title`: Defina o título da página, exibido na aba do navegador. 4. `meta`: Fornece metadados, como descrição da página e conjunto de caracteres. 5. `link`: Liga as folhas de estilo externas ao documento. 6. `body`: Contém todo o conteúdo visível da página. 7. `h1`, 'h2`, `h3`, ...
